# Person API

API provides a resource to people management.

## Tecnologies

* Java 11 +
* Spring Web
* Spring Data JPA
* H2 (Database)

## Resources

* people

| Method | Endpoint            |
| :----- |:------------------- |
| GET    | /api/v1/people      |
| GET    | /api/v1/people/{id} |
| POST   | /api/v1/people      |
| PUT    | /api/v1/people/{id} |
| DELETE | /api/v1/people/{id} |

## How to use

* On terminal: 

```bash
./mvnw spring-boot:run
```

* **Live**: [https://peopleapi-v1.herokuapp.com/](https://peopleapi-v1.herokuapp.com/)
